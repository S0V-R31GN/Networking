# Lab 01.3 – Connecting Devices Using Switch (Layer 2 – Setting Up a LAN)

## Switch vs Hub
A switch is network hardware that connects devices to establish a local area network (LAN).

**Switch pros (vs hub):**
- Has memory — stores a MAC address table
- Knows which port the destination device is on, and forwards data only to that port
- A hub has no memory/intelligence — it broadcasts incoming data to *all* ports regardless of destination

## MAC Address Table

| Device | MAC Address | Interface/Port |
|---|---|---|
| PC1 (sender) | AA-AA-AA-AA-AA-AA | 1 |
| PC2 (receiver) | BB-BB-BB-BB-BB-BB | 19 |

## Behavior
- Sender sends data to port 1.
- If this were a **hub**, it would broadcast the data to all other ports.
- A **switch** forwards the data only to the specified destination port (19).
