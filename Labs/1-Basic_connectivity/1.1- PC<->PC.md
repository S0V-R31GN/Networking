Lab 01.1 – Connecting 2 PCs

IP Assignment

PC1 IP: 10.10.10.1
PC2 IP: 10.10.10.2


Commands
Command prompt (run on each PC):

ipconfig
ipconfig /all

Shows IP address + MAC address.

On PC1, open command prompt:

ping 10.10.10.2

Shows the result of the packet sent.
