First, build 2 separate LANs, then connect them using a router. 3 devices on LAN 1, 3 devices on LAN 2.


LANNetworkDevicesRouter InterfaceLAN 1192.162.1.0/24PC0 (.10), PC1 (.11), PC5 (.12)Giga0/0 — 192.162.1.13LAN 2192.162.2.0/24PC4 (.10), PC3 (.11), PC2 (.12)Giga0/1 — 192.162.2.13

The cable from each LAN's switch to the router must use an IP address in that LAN's range — that's what Giga0/0 (192.162.1.13) and Giga0/1 (192.162.2.13) are.

Setup


Assign each PC a static IP within its LAN's subnet.
Set each PC's Default Gateway to its LAN's router interface IP (e.g. 192.162.1.13 for LAN 1 devices).
Turn on the router and enable each Giga0/x interface/port.
